# 📚 LearnLytics – Student Performance Analytics Dashboard

## Overview

LearnLytics is an interactive Student Performance Analytics Dashboard developed using Python, Pandas, Plotly, and Streamlit.

The project analyzes student academic performance data and provides meaningful insights through interactive visualizations and performance metrics.

---

## Features

✅ Attendance vs Score Analysis

✅ Grade Distribution Visualization

✅ Grade Percentage Analysis

✅ Top Student Performance Tracking

✅ Attendance Distribution Insights

✅ Interactive Dashboard Filters

✅ Real-Time Data Visualization
